Data Dojo is an interactive playground for manipulating data.
Use it for:
  • Simple everyday tasks like mapping a CSV to an SQL statement
  • Programming assessments or exercises
  • Preparing input data for Jupyter notebooks (then just load from CSV URL)
  • Building data pipeline prototypes with instant feedback
  • Learning Python or JavaScript
Examples:
  • Map & reduce product IDs from a CSV to an SQL "SELECT IN" statement
  • Transform hundreds of user sessions to a normalized (0 - 1.0) activity per user
  • Write data conditions (unit tests) for each step of a complex data pipeline
  • (Soon) Convert historical pricing data to a pandas DataFrame
